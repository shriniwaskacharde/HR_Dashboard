# HR Analytics Dashboard

## 📌 Project Overview
This project is an HR Analytics Dashboard created using Power BI.  
The dashboard helps analyze employee data such as:

- Employee Count
- Attrition Rate
- Department-wise Analysis
- Age Group Analysis
- Salary Analysis
- Job Satisfaction
- Gender Distribution

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset
- power query
- Data Visualization

## 📊 Dashboard Features
- Interactive Charts
- KPI Cards
- Filters & Slicers
- Department Analysis
- Employee Attrition Insights
